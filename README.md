## DEVELOPMENT :

1. Clone the Repo
   `https://github.com/abhisheknaiidu/Dev-s-Blog.git`
   
2. cd `Dev-s-Blog`

3. Install dependencies
   `npm install`
   
4. `npm run serve`

5. open `localhost:8081`
